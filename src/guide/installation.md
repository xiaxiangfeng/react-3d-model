# 安装

## 使用包管理器

我们推荐您使用包管理器([NPM](https://www.npmjs.com/), [Yarn](https://yarnpkg.com/), [PNPM](https://pnpm.io/))安装 react 3D Model。

使用 NPM:

```bash
npm install react-3dmodelx --save
```

使用 Yarn:

```bash
yarn add react-3dmodelx
```

使用 PNPM:

```bash
pnpm install react-3dmodelx
```
