# react-3d-model

> 加载 3D 模型的 react 组件。预览地址：https://xiaxiangfeng.github.io/react-3d-model/index.html#/

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father`,

```bash
$ npm run build
```
